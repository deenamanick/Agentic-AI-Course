# Practical 9.4 — Benchmark and Choose a Framework

## Practice

Run all three implementations against the same dataset. Compare quality, task completion, latency, model calls, cost, recovery, and implementation complexity.

Write an architecture decision record for a deterministic approval workflow, a collaborative research team, and a conversational simulation.

## Success checklist

- [ ] Measurements support every recommendation.
- [ ] Framework popularity is not used as evidence.
- [ ] The decision identifies migration and lock-in risks.
