# Practical 9.1 — Create a Framework-Neutral Contract

## Why

You cannot compare frameworks fairly if each version solves a different problem.

## Build

Define typed contracts for a three-stage research workflow: research, fact-check, and write. Keep prompts, tools, model settings, budgets, test cases, and expected output identical.

## Success checklist

- [ ] Business logic does not import a framework.
- [ ] Inputs, handoffs, and final output are typed.
- [ ] The shared evaluation set includes failure cases.
