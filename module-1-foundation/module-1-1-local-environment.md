# Practical 1.1 — Set Up the Local AI Environment

## Why

Local inference lets students learn the complete request flow without depending on a paid model API.

## Steps

1. Install Python 3.10 or newer and Ollama.
2. Create and activate a virtual environment.
3. Install `requirements.txt`.
4. Copy `.env.example` to `.env`.
5. Keep credentials out of source control.

## Success checklist

- [ ] Python and Ollama report their versions.
- [ ] Dependencies install without errors.
- [ ] The application imports successfully.
