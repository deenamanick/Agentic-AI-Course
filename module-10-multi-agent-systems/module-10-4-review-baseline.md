# Practical 10.4 — Review, Terminate, and Compare

## Build

Add a reviewer that may request one evidence-based revision. Then implement a single-agent baseline using the same tools and dataset.

## Measure

Compare groundedness, completeness, latency, tokens, and total model calls.

## Success checklist

- [ ] Review is limited to one revision.
- [ ] Termination is deterministic.
- [ ] Multi-agent complexity is retained only if it improves a defined metric.
