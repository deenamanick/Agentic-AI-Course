# Practical 10.1 — Design Roles and Handoff Contracts

## Why

Multiple vague agents create duplicated work. Specialized roles need explicit responsibilities and outputs.

## Build

Define researcher, fact-checker, and writer roles. Create typed handoffs containing task ID, findings, evidence, confidence, unresolved questions, and status.

## Success checklist

- [ ] Responsibilities do not overlap unnecessarily.
- [ ] Every claim can retain its evidence.
- [ ] Agents cannot delegate outside an allowlist.
