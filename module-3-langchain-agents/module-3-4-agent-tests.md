# Practical 3.4 — Test Tool Selection and Failures

## Practice

Test correct selection, unnecessary tool use, malformed arguments, tool exception, model timeout, and iteration-limit exhaustion. Mock tools where deterministic behavior is required.

## Success checklist

- [ ] Tests assert tool choice and arguments.
- [ ] A tool failure cannot crash the server.
- [ ] Iteration limits are covered.
- [ ] Final answers do not claim a failed action succeeded.
