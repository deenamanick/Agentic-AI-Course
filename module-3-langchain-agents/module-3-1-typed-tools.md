# Practical 3.1 — Create Typed Agent Tools

## Build

Inspect the calculator, timestamp, and echo tools. For each tool, document its purpose, arguments, return type, limitations, and side effects.

Add strict calculator input validation and reject unsupported expressions.

## Success checklist

- [ ] Tool names and descriptions are unambiguous.
- [ ] Inputs are validated before execution.
- [ ] Errors are safe and useful.
- [ ] No tool has unnecessary permissions.
