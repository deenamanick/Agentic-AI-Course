# Practical 13.2 — Build Operational Metrics

## Build

Create metrics for success rate, p50/p95 latency, cost per successful task, model errors, tool errors, retrieval retries, approval rate, and evaluation score.

## Success checklist

- [ ] Metrics distinguish user, model, tool, and system failures.
- [ ] Cost is attributed by workflow step.
- [ ] Dashboards avoid storing sensitive payloads.
