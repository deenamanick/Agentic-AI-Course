# Practical 12.1 — Threat Model an Agent System

## Build

Draw the trust boundaries among users, API, model, retrieved documents, tools, memory, database, and external services.

List assets, attackers, entry points, abuses, mitigations, and residual risk. Include prompt injection, data leakage, unauthorized actions, denial of wallet, and cross-user access.

## Success checklist

- [ ] Every external input is marked untrusted.
- [ ] Consequential actions are identified.
- [ ] Each high-risk threat has a testable mitigation.
