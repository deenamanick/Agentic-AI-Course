# Practical 8.4 — Citations, Injection Defense, and Evaluation

## Why

A citation is useful only when it supports the nearby claim and points to the real source.

## What you will build

Return answers with document and page citations. Add adversarial documents containing instructions such as “ignore previous rules.”

## Practice

Grade answer correctness, groundedness, citation correctness, and refusal on unanswerable questions. Keep retrieved content in a clearly marked untrusted-data boundary.

## Success checklist

- [ ] Every factual claim has supporting evidence.
- [ ] Citations resolve to the stored page.
- [ ] Retrieved instructions cannot alter system policy.
- [ ] The report includes answerable and unanswerable cases.
