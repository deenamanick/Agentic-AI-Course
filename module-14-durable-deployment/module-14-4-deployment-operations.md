# Practical 14.4 — Deploy and Operate the Workflow

## Build

Map the local FastAPI lab to a standalone Cloudflare Worker, Cloudflare Workflow, D1 job state, and optional R2 artifacts. Add rate limits, structured logs, and cleanup policies.

## Success checklist

- [ ] No Pages Functions are introduced.
- [ ] Deployment configuration documents required bindings.
- [ ] Cancellation and recovery are demonstrated.
- [ ] An operations runbook covers stuck and failed jobs.
