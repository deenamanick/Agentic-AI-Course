# Practical 11.3 — Deterministic and Model-Based Graders

## Build

Use deterministic checks for schemas, citations, required facts, forbidden actions, and budgets. Add an LLM judge only for qualities such as clarity or completeness.

Calibrate the judge against a human-reviewed sample and record disagreements.

## Success checklist

- [ ] Safety is never graded only by an LLM.
- [ ] Judge prompts and models are versioned.
- [ ] Human agreement is reported.
