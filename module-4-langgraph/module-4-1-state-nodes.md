# Practical 4.1 — Define Graph State and Nodes

## Build

Inspect the graph state and the `route`, `math_tool`, `plan`, `execute`, and `finalize` nodes. Document what each node reads and writes.

## Practice

Add validation at state boundaries and keep nodes focused on one responsibility.

## Success checklist

- [ ] State fields have clear types.
- [ ] Nodes return explicit updates.
- [ ] Internal state is not shared globally between requests.
